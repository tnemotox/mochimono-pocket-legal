App Review Notes

1. Overview
Mochimono Pocket is a checklist app designed for parents in Japan to manage their children's belongings for various occasions (school trips, pool days, daily commute, hospital visits, park outings, overnight trips, and sports days). The app supports family sharing with real-time sync so all family members can collaborate on checklists.

2. Backend & Cloud Sync
This app uses Firebase (Firestore) for cloud synchronization and data storage. An internet connection is required for account features and real-time family sharing. Data is stored in Firebase Firestore under the authenticated user's account.

3. Authentication
This app requires user authentication via Apple Sign-In or Google Sign-In. No demo account is provided â€” reviewers can sign in directly using their Apple ID or Google account. Authentication is required to access family sharing and cloud sync features.

4. No In-App Purchases or Advertisements
This version does not include any in-app purchases, subscriptions, or advertisements. All features are completely free.

5. Testing Instructions
- Launch the app
- On first launch, you will see a sign-in screen
- Sign in using Apple ID or Google account
- Complete initial registration (family member name and birthdate)
- From the home screen, tap "New Checklist" to create a checklist
- Select a scene (e.g., "School Trip")
- Select participating family members
- Set conditions (weather, temperature, transportation, duration)
- A checklist will be automatically generated with appropriate items
- Items are organized into "Night Before" and "Morning Of" tabs
- Check off items as you pack them
- You can add or remove items from the generated list
- View the "Diff" screen to see changes from the template
- Save customizations as "Family Rules" for future use
- To test family sharing: tap "Group" in settings, create a group, and share the invite code with another device
- Changes made on one device will reflect in real-time on other devices in the same group

6. Privacy
User data is stored in Firebase Firestore under authenticated accounts. Family information (children's names and birthdates) is associated with the user's account and shared only within the family group. No third-party analytics or advertising SDKs are included.

7. Target Audience
Parents and caregivers of infants, preschoolers, and elementary school children in Japan.

8. Language
The app UI is in Japanese only.
