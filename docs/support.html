サポート・お問い合わせ

もちものポケットの使い方や、よくあるご質問をまとめました。困ったときはこのページをチェックしてみてください。

使い方ガイド

はじめての方へ

アプリを開いたら、4つのステップですぐに使い始められます。スクリーンショット付きの詳しいガイドは今後追加予定です。

1
家族を登録する

アプリを開いたら、まずお子さんの名前と年齢を登録します。きょうだいがいる場合は人数分追加できます。

2
おでかけシーンを選ぶ

遠足、プール、旅行など7つのシーンから、今回のおでかけに合ったものをタップします。

3
条件を設定してリスト生成

参加する家族メンバーと、天気や移動手段などの条件を選びます。条件に合った持ち物チェックリストが自動で作成されます。

4
チェックして準備完了

「前日よる」と「当日あさ」に分かれたリストで、ひとつずつチェック。使うたびにアプリが我が家のルールを覚えて、次回はもっとラクになります。

FAQ

よくあるご質問

お問い合わせの前に、こちらもチェックしてみてください。

データはどこに保存されますか？

すべてのデータはお使いの端末内にのみ保存されます。サーバーへの送信やクラウド同期は一切行いません。アカウント登録も不要なので、安心してお使いいただけます。

アプリは無料ですか？

はい、もちものポケットは完全無料です。アプリ内課金もありません。

広告は表示されますか？

いいえ、広告は一切表示されません。お子さんと一緒に見る画面だからこそ、すっきりした画面を大切にしています。

インターネット接続は必要ですか？

いいえ、完全オフラインで動作します。電波のないキャンプ場や山間部でも問題なくお使いいただけます。

対応しているシーンを教えてください。

現在、以下の7つのシーンに対応しています。

  • 遠足・校外学習
  • プール（学校/園）
  • 通園・通学（通常日）
  • 通院
  • 公園（半日）
  • 旅行（1泊）
  • 運動会
通知が届きません。

iOSの設定で通知が許可されているか確認してみてください。

確認手順: 「設定」アプリ → 「通知」 → 「もちものポケット」 → 「通知を許可」がオンになっているか確認してください。

それでも届かない場合は、お手数ですがアプリを一度終了してから再度開いてみてください。

チェックリストを編集できますか？

はい、自由に編集できます。自動生成されたリストに対して、以下の操作ができます。

  • 持ち物の追加（「＋」ボタンから）
  • 持ち物の削除（スワイプまたは編集モードで）
  • 数量の変更（アイテムをタップして数量を調整）

編集した内容は「我が家ルール」として学習され、次回から自動で反映されます。

家族を追加・削除するには？

アプリの「設定」画面から操作できます。

追加: 設定 → 家族メンバー → 「メンバーを追加」をタップ → 名前と年齢を入力してください。

削除: 設定 → 家族メンバー → 削除したいメンバーを左にスワイプ → 「削除」をタップしてください。

「我が家ルール」とは何ですか？

チェックリストを使うたびに、アプリがご家庭の傾向を学習する仕組みです。

たとえば「紙コップはいつも外す」「替えの靴下はいつも追加する」といった操作を覚えて、次回のリスト生成時に自動で反映します。使えば使うほど、ご家庭にぴったりのリストになっていきます。

天気や交通手段の条件とは？

チェックリスト作成時に選べるオプション条件です。たとえば「雨の日」を選ぶと雨具やビニール袋が自動で追加されたり、「車移動」を選ぶとチャイルドシート関連のアイテムが提案されたりします。

条件に合わせてリストが変わるので、より正確な準備ができます。

「前日よる」「当日あさ」の2フェーズとは？

持ち物を「前の晩に準備するもの」と「当日の朝に用意するもの」に分けて表示する仕組みです。

リュックに詰めるものは前日の夜に、お弁当や水筒は当日の朝に。フェーズを分けることで「朝やること」が明確になり、バタバタが減ります。

アプリを削除するとデータは消えますか？

はい、アプリを削除するとすべてのデータが完全に消去されます。データは端末内にのみ保存されているため、クラウドからの復元はできません。

再インストールした場合は、最初からのスタートになります。

機種変更時にデータを引き継げますか？

現在のバージョンではデータの引き継ぎ機能はありません。今後のアップデートで対応を検討しています。

Android版はありますか？

現在はiOS版のみの提供です。Android版については今後の開発を検討中です。ご要望として承りますので、お気軽にご連絡ください。

要望や機能リクエストを送れますか？

もちろんです！「こんな機能がほしい」「ここが使いにくい」など、どんな小さなことでも大歓迎です。下記のお問い合わせ先までメールでお送りください。いただいたご意見は、アプリの改善に役立てさせていただきます。
